## 使用说明：

 1. 本项目使用gulp+require.js开发个人主页，项目中所有依赖库存放在node_modules下。
 2. 如需下载到本地使用npm install 安装模块依赖。
 3. 目前此项目在开发中，后续代码会持续更新。
